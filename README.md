# 🥐 Yoyobolen Backoffice

A comprehensive back-office management system specifically tailored for the Yoyobolen bakery business. This system integrates order tracking, financial modules, and multi-bank payment solutions.

## ✨ Features
- **Multi-Bank Payment Integration**: Dynamic virtual accounts for BCA, BNI, BRI, and Mandiri via Midtrans Core API.
- **Order Management**: End-to-end tracking of bakery orders.
- **Financial Module**: Secure and typed financial data handling.

## 💻 Tech Stack
- Framework: Next.js (App Router), TypeScript
- Payment Gateway: Midtrans Core API
- Database/ORM: Prisma
